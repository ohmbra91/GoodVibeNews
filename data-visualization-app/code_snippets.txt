Git Commands

Initialize Repository: git init
Add Files: git add .
Commit Changes: git commit -m "Your commit message"
View Commit History: git log
Roll Back to Previous Commit: git reset --hard <commit-hash>
Move Forward to Specific Commit: git reset --hard <commit-hash>
Add Remote Repository: git remote add origin https://github.com/yourusername/your-repo-name.git
Push Changes to GitHub: git push -u origin master
Clone Repository: git clone https://github.com/yourusername/your-repo-name.git
